Bitcoin Scan Bot v1.0Este bot realiza a varredura de chaves privadas em plataformas de indexação e verifica saldos na rede principal do Bitcoin.Configuração NecessáriaPara que o bot funcione corretamente, você deve configurar os seguintes Secrets no seu repositório GitHub (Settings > Secrets and variables > Actions):1.MASTER_KEY: Sua chave mestra ou senha de segurança.2.CUSTODY_ADDRESS: O endereço Bitcoin para onde os fundos encontrados serão enviados.Funcionamento•Frequência: O bot roda automaticamente a cada 5 minutos.•Continuidade: O estado da varredura é salvo em cache, garantindo que ele continue de onde parou no ciclo anterior.•Capacidade: Processa milhares de páginas por ciclo através de execução paralela.Estrutura de Arquivos•bitcoin_bot.py: Script principal em Python.•.github/workflows/main.yml: Configuração da automação do GitHub Actions.•bot_state.txt: Arquivo gerado automaticamente para controle de progresso.
